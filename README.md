# marathon-matches-manager

## for development

```commandline
git clone git@github.com:kryInit/marathon-matches-manager.git
cd marathon-matches-manager/front-end
npm install
npm run build
cd ../
poetry install
poetry run m3 --help
```

## for use
```commandline
pip install marathon-matches-manager
m3 --help
```
